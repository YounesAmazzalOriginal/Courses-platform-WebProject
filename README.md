# Plateforme de Cours en Ligne – Projet Web
Bonjour,
Ce projet a été réalisé dans le cadre du module de développement web semestre 2 informatique appliquée.

## 📌 Présentation du projet
Il s'agit d'une plateforme de cours en ligne permettant aux utilisateurs de consulter des formations, explorer les cours disponibles et accéder à leurs détails, avec des interfaces de connexion et d'inscription.

## 👨‍💻 Réalisation
Ce projet a été entièrement conçu et développé **individuellement**, sans travail en groupe.

## 🎨 Design
Le design de l’interface est **original et créé from scratch**.
La conception visuelle a été réalisée en amont à l’aide de Photoshop afin de définir la structure et l’idée générale du projet.

## 🧩 Pages du site
* **Accueil (index.html)** : page principale avec une présentation et quelques cours
* **Explorer les cours (explore-courses.html)** : affichage de tous les cours avec une fonctionnalité de recherche
* **Détails du cours (courses-details.html)** : informations complètes sur un cours
* **Connexion (login.html)** : interface de connexion
* **Inscription (signup.html)** : formulaire d’inscription
* **Contact (contact.html)** : page de contact

## ⚙️ Fonctionnalités
* Affichage dynamique des cours avec JavaScript
* Système de recherche en temps réel
* Interface utilisateur claire et moderne
* Validation basique des formulaires

## 🛠 Technologies utilisées
* HTML5
* CSS3
* JavaScript (Vanilla)
* Photoshop (pour la conception du design)
